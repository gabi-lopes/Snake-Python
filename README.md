# 🐍 Snake Game em Python  

Um clássico jogo da cobrinha desenvolvido em **Python** com a biblioteca **Pygame**.  

---

## 🎮 Demonstração  

A cobrinha se move pela tela, cresce ao comer a comida e o jogo termina se ela colidir com as paredes ou com o próprio corpo.

---

## ⚙️ Tecnologias utilizadas  

- **Python 3.x**  
- **Pygame** — biblioteca usada para criar jogos 2D
